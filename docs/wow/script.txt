- txt로 첨부한 내용은 게임 월드 오브 워크래프트의 기원사 직업 중 증강 특성에 대한 공략글이야.
- 웹페이지 내용을 그대로 텍스트로 긁었기 때문에 형식이 엉망인 상황이고 한글화가 일부 고유명사에만 되어있는 상태야.
- 이 내용을 원문 그대로 긁되, 표기 형식을 마크다운 형식에 적합하게 변환해줘.
- 대신 절대로 영어 원문을 임의로 한글로 번역하거나, 내용을 요약, 정리, 변경하지마. 모든 내용을 다 담아줘.
- 결과물은 plain text 블럭에 싼 raw한 md 문법으로 return 부탁해.
- [cite] 표기는 제거하고 return해줘.

- 첨부한 md 파일은 와우헤드에 올라와있는 증강 기원사 공략 글이야.
- 전체적으로 영문이고 일부 고유명사만 한글로 되어있어.
- 첨부한 csv 파일은 와우헤드에서 발췌한 기원사의 스킬 [영문명] - [한글명] 매핑 정보야.
- csv 내용을 참고하여 위 md 파일 내용을 한글로 자연스럽게 번역해줘.
- 번역은 하되, 절대로 내용을 임의로 요약, 정리, 변경하지 마. 모든 내용을 다 담아줘.
- 결과물은 plain text 블럭에 싼 raw한 md 문법으로 return 부탁해.
- [cite] 표기는 제거하고 return해줘.
